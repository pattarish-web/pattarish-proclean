## ไฟล์ในโฟลเดอร์นี้

| ไฟล์ | คำอธิบาย |
|------|----------|
| [SETUP.md](SETUP.md) | เช็คลิสต์ตั้งค่าเพจ + ค่าทุกช่องพร้อมแปะ |
| [copy-th.md](copy-th.md) | About + โพสต์เปิดเพจ + คำตอบรีวิว |
| [generate_assets.py](generate_assets.py) | สร้างรูปปกและโปรไฟล์ |
| `cover.png` | รูปปก 1640×856 |
| `profile.png` | รูปโปรไฟล์ 800×800 |

## สร้างรูปใหม่

จาก root ของ repo:

```bash
python marketing/facebook-page/generate_assets.py
```

## ทำต่อบนเพจ (จากภาพหน้าจอ — ยังว่าง)

1. อัปโหลด `profile.png` แล้ว `cover.png`
2. แปะ Bio / About จาก `SETUP.md` หรือ `copy-th.md`
3. ตั้งเว็บ เบอร์ เวลา CTA + username `sangkanclean`
4. โพสต์เปิดเพจอย่างน้อย 3 ชิ้นจาก `copy-th.md`
